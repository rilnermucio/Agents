# Gerar Prompt de Imagem IA

Crie um prompt otimizado para a ferramenta de IA especificada.

## Input do Usuario
$ARGUMENTS

## Ferramentas Suportadas

| Ferramenta | Melhor Para |
|------------|-------------|
| Midjourney | Arte conceitual, ilustracoes |
| DALL-E 3 | Versatilidade, edicao |
| Flux 2 Pro | Realismo, produtos |
| GPT Image 1.5 | Fotos reais |
| Nano Banana Pro | Posts prontos com texto |

## Estrutura de Prompt

```
[SUJEITO] + [ACAO] + [AMBIENTE] + [ESTILO] + [ILUMINACAO] + [CAMERA] + [QUALIDADE]
```

## Entregaveis

- Prompt principal otimizado
- 3 variacoes
- Parametros recomendados (aspect ratio, etc)
- Dicas de iteracao
