# Criar Email Marketing

Crie um email ou sequencia completa para o objetivo especificado.

## Input do Usuario
$ARGUMENTS

## Tipos de Email

| Tipo | Objetivo | Emails |
|------|----------|--------|
| Welcome | Onboarding | 3-5 |
| Nurturing | Relacionamento | 5-7 |
| Lancamento | Vendas | 7-10 |
| Carrinho | Recuperacao | 3 |

## Estrutura

```
SUBJECT LINE (max 50 chars)
PREVIEW TEXT (max 90 chars)
BODY
├── Hook de abertura
├── Corpo com valor
├── CTA claro
└── PS (opcional)
```

## Entregaveis

- Subject line + 2 variacoes
- Preview text
- Email completo formatado
- CTA
- Timing sugerido
