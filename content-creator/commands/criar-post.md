# Criar Post para Redes Sociais

Crie um post completo para a plataforma especificada.

## Input do Usuario
$ARGUMENTS

## Processo

1. **Identificar plataforma e formato** (feed, carrossel, reels, stories)
2. **Definir nicho e tom de voz**
3. **Criar conteudo** usando frameworks de copy (AIDA, PAS, BAB)
4. **Gerar hashtags** relevantes
5. **Sugerir horario** de publicacao

## Entregaveis

- Texto do post formatado para a plataforma
- 2-3 variacoes A/B
- Hashtags (mix de alcance: alto, medio, nicho)
- Horario sugerido
- Prompt de imagem IA (quando aplicavel)
