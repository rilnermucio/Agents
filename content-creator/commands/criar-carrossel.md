# Criar Carrossel Instagram

Crie um carrossel completo com estrutura de slides otimizada para engajamento.

## Input do Usuario
$ARGUMENTS

## Processo

1. **Definir tema e objetivo** (educativo, lista, storytelling, comparacao)
2. **Criar estrutura** de 7-10 slides
3. **Hook no slide 1** - capturar atencao
4. **Conteudo nos slides 2-9** - entregar valor
5. **CTA no slide final** - proxima acao

## Estrutura de Slides

```
Slide 1: HOOK (pergunta, afirmacao ousada, problema)
Slides 2-9: CONTEUDO (1 ideia por slide)
Slide 10: CTA (seguir, salvar, comentar)
```

## Entregaveis

- Texto de cada slide
- Sugestao de layout visual
- Legenda do post
- Hashtags
- Prompt de imagem IA para capa
