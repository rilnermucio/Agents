# Criar Calendario Editorial

Crie um calendario editorial completo para o periodo especificado.

## Input do Usuario
$ARGUMENTS

## Processo

1. **Definir pilares** de conteudo (3-5)
2. **Mapear datas** importantes do periodo
3. **Distribuir formatos** (feed, carrossel, reels, stories)
4. **Definir frequencia** por plataforma
5. **Criar grade** semanal/mensal

## Estrutura Semanal Sugerida

| Dia | Tipo | Objetivo |
|-----|------|----------|
| Segunda | Educativo | Valor |
| Terca | Storytelling | Conexao |
| Quarta | Lista/Dica | Engajamento |
| Quinta | Bastidores | Autenticidade |
| Sexta | CTA/Oferta | Conversao |

## Entregaveis

- Calendario completo (tabela)
- Temas por semana
- Formatos sugeridos
- Datas especiais mapeadas
- KPIs por tipo de conteudo
