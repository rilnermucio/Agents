# Criar Roteiro de Video

Crie um roteiro completo para o formato de video especificado.

## Input do Usuario
$ARGUMENTS

## Formatos Suportados

| Formato | Duracao | Uso |
|---------|---------|-----|
| YouTube Long | 8-20 min | Tutorial, educativo |
| Reels/TikTok | 15-60s | Viralidade |
| Shorts | 15-60s | Descoberta |
| VSL | 5-30 min | Vendas |

## Estrutura Base

```
HOOK (0-3s): Ganhar atencao
SETUP (3-30s): Prometer valor
CONTEUDO: Entregar valor
CTA: Proxima acao
```

## Entregaveis

- Hook (primeiros 3 segundos)
- Roteiro completo com timestamps
- Direcoes visuais (B-roll, texto na tela)
- Sugestao de thumbnail
- CTA final
