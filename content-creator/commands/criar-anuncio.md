# Criar Anuncio

Crie copy de anuncio otimizado para a plataforma especificada.

## Input do Usuario
$ARGUMENTS

## Plataformas

| Plataforma | Formatos |
|------------|----------|
| Meta Ads | Imagem, Video, Carrossel, Stories |
| Google Ads | Search, Display, YouTube |
| TikTok Ads | In-Feed, TopView, Spark |
| LinkedIn Ads | Sponsored Content, InMail |

## Limites de Caracteres (Meta)

| Campo | Limite |
|-------|--------|
| Primary Text | 125 chars |
| Headline | 40 chars |
| Description | 30 chars |

## Entregaveis

- 3-5 variacoes de copy
- Headlines testados
- CTAs por objetivo
- Sugestao de publico
- Formato recomendado
