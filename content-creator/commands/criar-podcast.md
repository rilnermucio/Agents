# Criar Roteiro de Podcast

Crie um roteiro estruturado para o episodio especificado.

## Input do Usuario
$ARGUMENTS

## Formatos

| Formato | Descricao |
|---------|-----------|
| Solo | Apresentador unico |
| Entrevista | Host + convidado |
| Co-host | 2+ apresentadores |
| Storytelling | Narrativa |

## Estrutura

```
INTRO (30s-2min)
├── Hook de abertura
├── Apresentacao do tema
└── Promessa de valor

CONTEUDO (80% do tempo)
├── Segmento 1
├── Segmento 2
├── Segmento 3
└── Intervalo (se houver)

FECHAMENTO (2-5min)
├── Resumo
├── CTA
└── Preview proximo
```

## Entregaveis

- Roteiro estruturado
- Timestamps sugeridos
- Show notes
- Sugestoes de titulo
- CTA para audiencia
