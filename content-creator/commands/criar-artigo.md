# Criar Artigo SEO

Crie um artigo completo otimizado para ranquear no Google.

## Input do Usuario
$ARGUMENTS

## Processo

1. **Keyword research** - principal e secundarias
2. **Estruturar outline** - H1, H2, H3 otimizados
3. **Escrever conteudo** - seguindo E-E-A-T
4. **Otimizar on-page** - title, meta, URLs
5. **Validar checklist** SEO

## Entregaveis

- Artigo completo formatado
- Title tag (max 60 chars)
- Meta description (max 155 chars)
- URL slug sugerida
- Keywords utilizadas
- Sugestoes de internal linking
