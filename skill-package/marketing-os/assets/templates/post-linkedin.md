# Template: Post LinkedIn

## Informações do Template
- **Formato**: Post de texto (com ou sem imagem)
- **Limite**: 3.000 caracteres (140 visíveis no preview)
- **Hashtags**: Máximo 5

---

## ESTRUTURA DO POST

```
[HOOK - PRIMEIRA LINHA]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Uma frase impactante que aparece no preview.
Ela precisa fazer a pessoa clicar em "ver mais".

Use ponto final e quebra de linha para criar suspense.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[CORPO DO POST]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[Contexto/Setup]
Apresente a situação, história ou problema.

[Desenvolvimento]
Desenvolva seu ponto principal.
Use parágrafos curtos.
Cada um com 2-3 linhas no máximo.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## EXEMPLOS DE HOOKS

### Para Histórias
```
Há 3 anos, eu estava no pior momento da minha carreira.
↳
Recebi uma ligação que mudou tudo.
↳
"Você está demitido" — e foi a melhor coisa que me aconteceu.
```

### Para Lições
```
Depois de 10 anos em [área], aprendi uma verdade dura:
↳
O maior erro que vejo profissionais cometendo é [X].
↳
Ninguém te conta isso sobre [tema], mas deveria:
```

### Para Opinião
```
Opinião impopular: [opinião controversa]
↳
Discordo de [consenso comum]. Aqui está o porquê:
↳
Todo mundo fala de [X], mas ignora [Y].
```

### Para Dados
```
[Dado surpreendente].
↳
Passei [tempo] analisando [X]. Os resultados me chocaram.
↳
A maioria acredita [X]. Os dados mostram [Y].
```

## FORMATOS QUE PERFORMAM

### 1. Post de História (Storytelling)
```
[Situação inicial]
↳
[Conflito/Desafio]
↳
[Virada/Descoberta]
↳
[Resolução]
↳
[Aprendizado aplicável]
```

### 2. Post de Lista
```
[Número] coisas que aprendi sobre [tema]:

1. [Ponto] - [Explicação breve]
2. [Ponto] - [Explicação breve]
3. [Ponto] - [Explicação breve]

Qual ressoou mais com você?
```

### 3. Post de Contraste
```
[Como a maioria faz X]
vs
[Como os melhores fazem X]

A diferença está em [insight].
```

### 4. Post de Pergunta
```
Pergunta genuína para minha rede:

[Pergunta]

Contexto: [Por que você está perguntando]

Agradeço quem puder compartilhar sua perspectiva.
```

## CTAs QUE FUNCIONAM

```
PARA ENGAJAMENTO:
• Qual foi sua experiência com isso?
• Concorda ou discorda? Comenta aqui.
• O que você acrescentaria a essa lista?

PARA CONEXÃO:
• Me conta nos comentários: [pergunta específica]
• Se isso fez sentido, compartilha com alguém que precisa ler.
• Marca alguém que está passando por isso.

PARA AUTORIDADE:
• Quer se aprofundar? Comenta [palavra] que te envio [recurso].
• Segue para mais conteúdo sobre [tema].
```
