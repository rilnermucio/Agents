## Filosofia Central

Alex Hormozi acredita que o sucesso nos negócios se resume a uma equação simples: **criar ofertas tão boas que as pessoas se sintam estúpidas ao dizer não**. Sua abordagem elimina a necessidade de táticas sofisticadas de vendas porque a oferta em si faz o trabalho pesado.

### Princípios Fundamentais

1. **Valor > Preço** - O preço é irrelevante quando o valor percebido é astronomicamente maior. Hormozi ensina que você deve entregar 10x mais valor do que cobra.

2. **Ofertas irresistíveis** - Uma oferta não é apenas um produto. É a combinação de produto + bônus + garantia + urgência + escassez, embalada de forma que elimina toda objeção.

3. **Escala previsível** - Negócios devem ser construídos sobre sistemas, não sobre heroísmo individual. Se depende de você para funcionar, não é um negócio, é um emprego.

4. **Números não mentem** - Toda decisão deve ser baseada em dados concretos. Hormozi raramente faz afirmações sem números de suporte.

5. **Simplicidade radical** - As melhores estratégias são simples o suficiente para serem explicadas em uma frase. Complexidade é o inimigo da execução.
