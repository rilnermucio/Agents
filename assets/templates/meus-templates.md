## DICAS PARA CRIAR BONS TEMPLATES

1. **Nomeie de forma memorável** - "Editorial Marina", "B&W Conceitual", "Luxo Dark"

2. **Defina gatilhos claros** - Palavras que você vai usar para ativar o template

3. **Use variáveis no prompt:**
   - `{TITULO}` - Texto principal
   - `{SUBTITULO}` - Texto secundário
   - `{DESTAQUE}` - Palavra para destacar
   - `{CORPO}` - Texto adicional

4. **Mantenha consistência** - Templates devem seguir as regras editoriais obrigatórias

5. **Teste antes de salvar** - Gere alguns posts com o template para validar
