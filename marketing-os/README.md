## Skills

| Skill | When Used |
|-------|-----------|
| marketing-os | Orchestrates all subagents for complete content creation |
| copy-agent | Headlines, CTAs, persuasive copy, A/B variations |
| seo-agent | SEO optimization, keywords, meta tags |
| social-agent | Social media posts, hashtags, timing |
| video-agent | Video scripts, hooks, retention strategies |
| audio-agent | Podcast scripts, audio content |
| ai-tools-agent | AI image/video prompts (Midjourney, DALL-E, etc.) |
| design-agent | Visual direction, color palettes, typography |
| analytics-agent | Metrics, KPIs, performance analysis |
| email-agent | Email sequences, newsletters |
| ads-agent | Ad copy for Meta, Google, TikTok |
| research-agent | Trends, competitor analysis, audience research |
